How To

The first thing I did was link a css page to my page.

        <!DOCTYPE html>
<html>
<head>
<title></title>
<link rel="stylesheet" href="howto.css" type="text/css" />
</head>
<body>
    
    <h1 id="title">reclassified</h1>

I put an open html tag to start the page off. I then inserted an open head tag to start the title at the top. After that I created a css page where I added details to decorate the title. I put a body tag there and then I added my first header. I used an id to be able to modify the text size and font on css. I then closed the header tag.


The second thing I did was my navigation bar.

            <ul id="navbar">
    
    <li><a href="Home.html">Home</a></li>
    <li><a href="index.html">About Me</a></li>
    <li><a href="hardware.html">Hardware</a></li>
    <li><a href="calculator.html">Calculator</a></li>
    <li><a href="howto.html">How To</a></li>
    <li><a href="#">Vindinium</a></li>
    <li><a href="bigproject.html">Big Project</a></li>

</ul>

I added a UL tag, which is an unordererd list. "a href" is basically a link. I then linked my different pages to the navbar and gave them a title. Then closed the ul tag.




The third step was making my header

        <h1 id="header">About Me</h1>

As you can see I used h1, which is the biggest header there is. And I put About Me, because that is what the page is about.
