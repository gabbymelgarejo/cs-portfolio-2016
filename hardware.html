Hardware






Motherboard: connects everything together, it is basically the hub of the computer.







CPU: stands for Central Processing Unit. It processes all the data coming through.









RAM: stands for Random Access Memory. It is where all the currently accessable files and have short term memory.










Hard Drive: basically storage for all the files, that can be retrieved at any time.










BUS: where all the information is transferred to the motherboard. It is the highway of information.










Computer: Keyboard- it is literally a board with keys that you use to type with. It includes the letters of the alphabet and numbers and symbols. Mouse- The thing that moves the pointer on a computer. Moniter- A tv like thing, that displays things.
